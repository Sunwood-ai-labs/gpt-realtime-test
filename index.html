オペレーション Tips

  • 環境ノイズ対策

    環境音が大きいときは navigator.mediaDevices.getUserMedia の音声オプションに noiseSuppression を追加すると雑音が軽減できます。

    navigator.mediaDevices.getUserMedia({
  audio: {
    echoCancellation: true,
    noiseSuppression: true
  }
});
  • /token レスポンスの期待値

    API 側では下記のように value プロパティへエフェメラルキーを格納する実装を想定しています。

    {
  "value": "EPHEMERAL_KEY"
}
  • DataChannel の活用

    DataChannel に届くイベントはログにも流れます。bot 側から送るメタ情報やアラートを受け取ることで、運用対応を素早く行えます。

    例: ステータス更新、ユーザー通知、フェイルオーバー指示などを JSON メッセージで送信

リモート音声ストリーム

Remote audio は WebRTC の track として受信され、ここで直接モニタリングできます。

リアルタイムログ

接続・エラー・イベントの情報がタイムスタンプ付きで記録されます。


                    


                    

                        

                            
                            セキュア接続メモ
                        

                        
Realtime API へのアクセスは HTTPS + TLS1.2 以上が推奨。ブラウザ側で Mixed Content
                            が発生しないよう注意し、証明書期限にもリマインダーを設定しましょう。
